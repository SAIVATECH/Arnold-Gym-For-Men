# Arnold Gym For Men (அர்னால்ட் ஜிம் போர் மென்)

A professional, high-conversion gym website built with Next.js, Tailwind CSS, and Framer Motion for **Arnold Gym For Men**, Kovilpatti.

## Features
- ✅ **High-Conversion Design:** Optimized for mobile users with a sticky action bar (Call/WhatsApp).
- ✅ **Interactive UI:** 3D tilt-on-hover effects, parallax backgrounds, and smooth scroll animations.
- ✅ **Lead Generation:** WhatsApp-integrated enquiry form that captures Name, Phone, and Fitness Goals.
- ✅ **Tamil/English Support:** Bilingual content for better local reach.
- ✅ **Performance Optimized:** Built with Next.js 15 App Router for lightning-fast loading.

## Sections
- **Hero:** Fullscreen with animated stats and dual CTAs.
- **About:** Hardcore environment highlights with Tamil translations.
- **Features:** Interactive cards showing gym facilities.
- **Transformation:** 3-stage journey overview.
- **Reviews:** Auto-sliding testimonial carousel.
- **Gallery:** Responsive image grid with full-screen lightbox.
- **Timings:** Opening hours and location map.

## Tech Stack
- **Framework:** [Next.js 15](https://nextjs.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)

## Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/SAIVATECH/Arnold-Gym-For-Men.git
   cd Arnold-Gym-For-Men
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Repository Structure
- `src/app`: Root layout, page, and global styles.
- `src/components`: Reusable UI sections and interaction logic.
- `public`: Static assets.
